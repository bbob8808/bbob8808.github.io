한국사능력검정 1급 - 근현대사 핵심정리

1. 개항기 흐름 (1860s~1890s)

● 흥선대원군의 개혁

왕권 강화 정책: 서원 철폐, 경복궁 중건, 비변사 폐지

통상 수교 거부: 병인박해 → 제너럴셔먼호 사건 → 병인양요 → 신미양요

“이 시기 핵심은? 외세 배척 + 왕권 회복!”

● 개항과 조약 체결

1876년 강화도 조약: 최초의 근대적 불평등 조약

→ 조일수호조규부록, 조일무역규칙 포함

● 개화 정책과 반발

수신사/영선사/보빙사 파견, 별기군 창설 → 임오군란 발생

→ 제물포조약, 청 내정 간섭, 갑신정변(1884): 3일천하

2. 근대 국가 수립 운동

● 동학농민운동

1차 봉기(반봉건) → 전주화약 → 2차 봉기(반외세)

→ 전봉준 체포, 일본 영향력 강화

● 갑오·을미개혁

갑오개혁: 신분제·과거제 폐지, 탁지아문

을미개혁: 단발령, 태양력 → 을미의병 발생

● 독립협회 활동

서재필, 독립신문, 만민공동회 → 헌의 6조 발표

3. 국권 피탈과 항일 의병

● 을사늑약과 통감부

1905년 을사늑약: 외교권 박탈, 통감부 설치

→ 을사의병, 시일야방성대곡 발표

● 국채보상·계몽운동

국채보상운동: 대구 시작, 전국 확대

애국 계몽운동: 보안회 → 대한협회 등 실력양성 노선

4. 기출 선지 분석 예시

문제: 갑신정변의 개혁 내용으로 옳지 않은 것은?
① 문벌 폐지
② 내시부 폐지
③ 순사 제도 설치
④ 양전 사업 실시
정답: ④
→ 양전 사업은 동학농민운동의 요구 사항으로 갑신정변과 무관함

복습 요약

  • 외세 침투 → 의병 활동 → 애국 계몽운동
